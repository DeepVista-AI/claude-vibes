How it works

Claude Code fires lifecycle hooks as it works. claude-vibes listens and plays a random voice clip for each event.

1

Install

One command copies 26 sound files to ~/.claude/vibes and registers hooks in Claude Code's settings.json.

2

Hook

Claude Code fires Stop, PermissionRequest, and Notification events as it works.

3

Play

Each hook triggers a script that picks a random .mp3 from the matching category and plays it with afplay.

Claude Code → Hook event → play-random.sh → Random voice clip
